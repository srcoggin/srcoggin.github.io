# GreenEighteen Sports (GES)

![GreenEighteen Sports](https://srcoggin.github.io/Logo.png)

**Real-time NFL coverage**

GreenEighteen Sports (GES) is a modern NFL news aggregator and analytics platform. It curates high level content from top sports networks and leverages AI to provide concise, real time summaries and insights for football fans, fantasy players, and analysts.
